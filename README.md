# shinycolors-background-sound

アイドルマスター シャイニーカラーズのPCブラウザ版で、ウィンドウからフォーカスが外れた状態でも音声の再生を継続させるスクリプトです。
他の方のスクリプトがシャニマスの仕様変更？で使えなくなっていたので作成しました。

## インストール方法
1. Tampermonkey等のユーザースクリプトツールをインストール

https://www.tampermonkey.net

ブラウザの設定で、デフォルトではユーザースクリプトの実行が許可されていない場合があります。許可する設定にしてください。
1. 本スクリプトをインストール

https://github.com/Hajime0512/shinycolors-background-sound/raw/main/shinycolors-background-sound.user.js

1. スクリプトを有効化

## 使い方
特に設定などはなく、スクリプトが有効化されていれば機能します。
うまく動かない場合、シャニマスのページを再読み込みしたり、新しいタブで開いたりしてみてください。

ミュートしたいときは、タブのミュート機能などを使用すると良いと思います。
